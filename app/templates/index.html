Welcome to Word Rush!

This is a typing based game.

You should definitely make an account to unlock daily mode!

How to play!

You need to type as many words as you can in a minute! However, the words need to contains specified letters in order to be counted.

For example: The letters TH
POOL = 0pt (doesn't have T or H)
HER = 35pt (contains a H so a base of 10*(number of letters)pt + 5 for each use of the specified letters)
HATCH = 65pt (valid word, so base of 50pt. Plus 15pt for 3 occurrence of T or H)
THAT = 110pt (you get a base of 40pt, plus 15 for using T or H 3 times. BUT since TH are together you get a 2 times multiplier, 55*2 equal 110)
